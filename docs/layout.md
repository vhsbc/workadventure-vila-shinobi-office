# Layout do Prototipo

Nome interno: `Vila Shinobi Office`

## Zonas

- `Sala principal`: ambiente unico com 10 estacoes individuais
- `Entrada`: acesso principal centralizado
- `Circulacao interna`: faixa livre entre as duas fileiras de mesas

## Intencao do layout

- As `10 estacoes` ficam dentro de uma unica sala para simplificar o uso diario.
- O layout usa `duas fileiras de 5 mesas`, deixando a leitura do espaco imediata.
- A `circulacao central` permite que o avatar atravesse a sala sem cruzar por dentro dos postos.
- A `entrada central` facilita o spawn e o onboarding de quem entra no mapa.

## Comportamentos embutidos

- `start`: area de entrada logo acima da porta principal
- `collides`: aplicado em paredes, mesas e cadeiras

## Observacoes

- O tileset atual e propositalmente simples e original para acelerar iteracao.
- Esta versao reduz o prototipo ao minimo funcional antes de uma nova rodada visual.
- A proxima etapa natural e melhorar a linguagem grafica para algo mais clean e moderno.
